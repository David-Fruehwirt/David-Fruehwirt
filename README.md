EmberExchange is an economy and collection game built around virtual stoves. Players open lootboxes, collect stoves with different rarities, forge new items, and trade with other players.

The marketplace uses a real coin economy. Players can follow price history, check who owned an item before, and build a collection over time. Real-time Poker, Blackjack, and Roulette add more ways to play.

My work focused on the Angular frontend and the middle layer that connects the interface to services and shared TypeScript models.
