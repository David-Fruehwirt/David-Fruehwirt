**Frontend Developer · Middle-Layer Contributor**

### [RedQuorum](https://github.com/LPuehringerStudent/RedQuorum)

**Co-Creator · General Contributor · First-Place Project**

<picture>
  <source media="(prefers-color-scheme: dark)" srcset="https://raw.githubusercontent.com/David-Fruehwirt/David-Fruehwirt/main/assets/redquorum-dark.svg">
  <source media="(prefers-color-scheme: light)" srcset="https://raw.githubusercontent.com/David-Fruehwirt/David-Fruehwirt/main/assets/redquorum-light.svg">
  <img alt="RedQuorum security pipeline showing scanning, agent debate and consensus" src="https://raw.githubusercontent.com/David-Fruehwirt/David-Fruehwirt/main/assets/redquorum-light.svg">
</picture>

A multi-agent AI security scanner where specialized agents inspect potential vulnerabilities, debate their impact, and vote. Only findings that reach quorum are promoted to the final report.

Built by a creator group and awarded **First Place at Youth Entrepreneurship Week in Linz**.
